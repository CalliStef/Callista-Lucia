# Adding Content in Figma
{: .no_toc }

While not a feature unique to Figma, it is important to be able to add actual context to our application mockup through text boxes! We will be using the Text tool in Figma to add text boxes of different colors and sizes.

## Task 1 - Entering Text
**1.1** _Click_ on the "T" icon on the **Navigation Menu** above our workspace. This is the **Text** tool and will turn blue once selected, as shown below. You should also notice your cursor turning into a cross-hair shape, rather than an arrow.

<img src="https://github.com/CalliStef/Callista-Lucia/blob/gh-pages/assets/images/task2.1_img1.png?raw=true" width="450" />
</div>
<br>

**1.2** _Click_ anywhere inside the â€œLoginâ€ frame. This will paste your text box into the frame. You can check you have done this correctly by looking on the **Layers Panel** on the left-hand side and seeing your "Login" frame have a new **Text Layer** indented below it. 

<div align="center">
<img src="https://github.com/CalliStef/Callista-Lucia/blob/gh-pages/assets/images/task2.1_img2.png?raw=true" width="500" />
